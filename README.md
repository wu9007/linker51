# Linker51 视觉实验项目
本实验项目主要实现机械臂的单目视觉定位功能，通过图像识别确定目标物体并将其坐标同步至机器人坐标系。

## 实验硬件清单 (BOM)
- 执行器：
    - 底座舵机：MG996R（大扭矩，负责水平旋转） 
    - 肩部舵机：MG90S（金属齿轮，负责大臂升降） 
    - 轴部舵机：SG90（轻量化，负责末端/小臂）
- 主体结构：矿泉水瓶 ×3（用于搭建支撑杆与机械臂连杆）
- 控制核心：普中 A2 单片机（51内核）
- 视觉与计算：
    - 单目摄像头（全局视野俯拍）
- 电脑：Windows / Mac
- 供电系统：
    - 充电宝（作为独立动力源） 
    - DIY 电源线：USB 剪线（红正黑负，接入面包板公用正负极，实现与开发板共地）
- 连接件：面包板、数据线、杜邦线
